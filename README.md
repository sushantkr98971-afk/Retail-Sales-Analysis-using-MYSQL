# Retail-Sales-Analysis-using-MYSQL
The repository contains SQL queries that solve common Data Analyst Problems using a tbale called sales_details each query demonstrates filtering, aggregation, grouping and window function to answer business questions.

These queries demonstrate the core responsibilities of a data analyst:
1. Filtering and cleaning data
2. Aggregating metrics (totals, averages, counts)
3. Grouping by categories, time, or demographics
4. Using window functions for ranking and comparisons
5. Translating business questions into SQL queries
